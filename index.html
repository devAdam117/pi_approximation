π test

Let X has uniform distribution R(0,1) By this it means that X generates random number between 0 and 1 , by this value we want to calculate approximate value of π. Effective way is to generate random points across unit square. Every generated point will have this pattern= (x,y) which actually is (X~R(0,1), X~R(0,1)) How many points you want to generate in the square?

  • Red coloured are oppoiste ones
  • The more points you generate the more the green area of points looks like a quarter of a unit circle
  • Area of quarter unit circle is: \[ { π 1^2\over4}\]
  • Area of unit square is: \[ { 1 }\]
  • (Area of quarter circle/Area of unit square)*4 is: π
